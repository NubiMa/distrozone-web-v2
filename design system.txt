STREETWEAR DISTRO – DESIGN STRUCTURE
===================================

[ COLOR PALETTE ]
----------------
Primary (Base Black)        : #0F0F0F
Background (Main White)     : #FFFFFF
Background Secondary        : #F5F5F5
Accent (Street Orange)      : #FF6B00
Text Primary                : #111111
Text Secondary              : #9CA3AF
Border / Divider             : #E5E7EB


[ GLOBAL RULE ]
-------------
- Dominant color            : White (70%)
- Primary black usage       : Header, Footer, Button
- Accent usage              : CTA, Price, Badge ONLY
- Max accent color          : 1
- No gradient / neon
- White space               : Large


[ HEADER / NAVBAR ]
------------------
Background                  : #0F0F0F
Logo                         : #FFFFFF
Menu text (default)          : #E5E7EB
Menu text (hover/active)     : #FF6B00
Search icon                  : #FFFFFF
Cart icon                    : #FFFFFF
Bottom border                : none


[ HERO SECTION ]
---------------
Background                   : Product image / White
Headline text                : #111111
Subheadline                  : #6B7280
Primary CTA button            :
    - Background             : #0F0F0F
    - Text                   : #FFFFFF
    - Hover background        : #FF6B00
Secondary CTA (optional)     :
    - Border                 : #0F0F0F
    - Text                   : #0F0F0F


[ CATEGORY SECTION ]
-------------------
Section background            : #FFFFFF
Category card background      : #F5F5F5
Category title                : #111111
Hover effect                  : Border #FF6B00


[ PRODUCT CARD ]
---------------

Product name                  : #111111
Price                         : #FF6B00
Old price (if sale)           : #9CA3AF

    - Background              : #0F0F0F
    - Text                    : #FFFFFF
    - Hover                   : #FF6B00
Badge (NEW / SALE)            :
    - Background              : #FF6B00
    - Text                    : #FFFFFF

    - Hover                   : #FF6B00

----------

[ TYPOGRAPHY ]
-------------
Heading (H1–H3)               : Bold
Font recommendation           : Inter / Satoshi / Space Grotesk
Body text                     : Regular
Text color                     : #111111


[ FINAL NOTES ]
-------------
- No colorful illustrations
- Focus on product photography
- Street vibe comes from layout & spacing
- Simple ≠ boring, detail = character
